# Smaller

## Constructors

|Constructor|Description|
|----|----|
|`carbon.Smaller(args ...any)`|Returns a `SMALL` text view for compact supporting inline content.|

## Basic Usage

```go
copy := carbon.Para(
 "Build 218 ",
 carbon.Smaller("experimental"),
)
```
