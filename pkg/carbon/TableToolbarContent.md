# TableToolbarContent

## Constructors

|Constructor|Description|
|----|----|
|`carbon.TableToolbarContent(args ...any)`|Returns a `cds-table-toolbar-content` wrapper for action controls inside a toolbar.|

## Basic Usage

```go
actions := carbon.TableToolbarContent(
 carbon.Button("Add row"),
)
```
