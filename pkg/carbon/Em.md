# Em

## Constructors

|Constructor|Description|
|----|----|
|`carbon.Em(args ...any)`|Returns an `EM` text view for emphasized inline content.|

## Basic Usage

```go
copy := carbon.Para(
 "This setting is ",
 carbon.Em("recommended"),
 " for most deployments.",
)
```
