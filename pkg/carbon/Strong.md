# Strong

## Constructors

|Constructor|Description|
|----|----|
|`carbon.Strong(args ...any)`|Returns a `STRONG` text view for strongly emphasized inline content.|

## Basic Usage

```go
copy := carbon.Para(
 carbon.Strong("Important:"),
 " Confirm the deployment window before release.",
)
```
