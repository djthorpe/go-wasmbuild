# Icons

Icons are rendered using [Bootstrap Icons](https://icons.getbootstrap.com/). You can search for
specific icons below. Create an icon by name using `bs.Icon()`. Options for text formatting include:

* Sizing using Bootstrap font size utility classes (e.g. `fs-1`, `fs-2`, etc.)
* `bs.WithColor(bs.Color)` - set the icon color using Bootstrap color classes
