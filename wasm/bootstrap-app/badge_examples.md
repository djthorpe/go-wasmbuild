
# Badges

Badges scale to match the size of the immediate parent element by using relative font sizing. There
are two types of badges: regular badges created with `bs.Badge()` and pill badges created with `bs.PillBadge()`, which
are more rounded.

Options for badges include:

* `WithColor(bs.Color)` - set the background color

The [Bootstrap documentation on Badges](https://getbootstrap.com/docs/5.3/components/badge/) provides more details and examples.
